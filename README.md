# kpw2_lora
LORA chipset receive and show data with Kindle PaperWhite 2


## Kindle PaperWhite 2

Hardware Version: Kindle 5.4.3.2 (2317380003)

JailBreak

USBNetwork

```shell
ssh root@192.168.31.150


Welcome to Kindle!

#################################################

[root@kindle root]# 


```

**NOTE** <***mntroot rw***> will make rootfs writable
